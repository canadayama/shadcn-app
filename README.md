# Shadcn UI Crash Course

## Install Next.js App

`$ npx create-next-app@latest shadcn-app --typesecript --tailwind --eslint`

## Install Shadcn-ui

`$ npx shadcn-ui@latest init`

## Json Server

### Install Json Server

`$ npm install -g json-server`

### Start Json Server

`$ json-server --watch ./_data/db.json --port 4000`
